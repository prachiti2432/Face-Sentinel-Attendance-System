# Restore Face Sentinel Project

## To reopen the project in Cursor

1. **File → Open Folder** (or `Ctrl+K Ctrl+O`)
2. Navigate to: `C:\Users\manth\Desktop\FinalYearProject\Face-attendance-system`
3. Click **Select Folder**

## To run the project

```bash
cd C:\Users\manth\Desktop\FinalYearProject\Face-attendance-system
npm run dev
```

Then open **http://localhost:5173** in your browser.

## Camera pages

- **Register**: http://localhost:5173/register.html
- **Attendance**: http://localhost:5173/attendance.html

## If camera still doesn't work

1. **HTTPS required**: Camera needs a secure context. Use `localhost` or HTTPS.
2. **Grant permission**: Allow camera access when the browser prompts.
3. **Close other apps**: Ensure no other app is using the webcam.
4. **Check console**: Press F12 → Console to see any errors.
